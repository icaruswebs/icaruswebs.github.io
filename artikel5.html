Perbedaan Game 8 Bit Jadul dan Modern

> by Blog Xoupedia

> Mei 02, 2022

> Edukasi


Game yang memiliki grafik dengan ciri khas gambar mozaik, berpiksel besar, dan memiliki susunan warna yang terbatas. Sebut saja game 8 bit atau game yang menggunakan berbagai jenis sprite berbasis pixel art. Hal ini terkadang cukup menuai pro dan kontra di kalangan para gamer. Terutama dikalangan gamer generasi sekarang yang lebih mengutamakan grafik game yang hd, dibandingkan dengan gameplay.

Hal yang seperti demikian bisa kita temukan didalam berbagai jenis komentar yang ada di media sosial. Banyak orang yang mengakunya gamer, yang selalu saja menghina grafik game yang tidak terlalu bagus. Dan menyebut game tersebut sebagai game 8 bit. Sebagai contoh korban dari ejekan para netizen ini tentu misalnya adalah game free fire dan mobile legends.


Apakah Game Berbasis Pixel Art dapat disebut Sebagai Game 8 Bit ?

Muncul sebuah pertanyaan unik di benak kita. Apakah game yang memiliki grafik jelek dapat disebut sebagai game 8 bit ?. Jawabannya tentulah tidak, karena jika yang dimaksud dengan game 8 bit adalah style game dengan ciri khas game konsol jadul ala nintendo nes dan sega di masa lalu. Maka game seperti mobile legend, free fire dan teman-temannya sama sekali tidak dapat disebut sebagai game 8 bit.

Karena ciri khas dari game 8 bit yang sesuai dengan gaya game retro, di haruskan memiliki grafik game bergaya pixel art. Baik itu game 2D maupun game 3D.

selanjutnya Apakah game berbasis pixel art dapat disebut sebagai game 8 bit ?. Jawaban dari pertanyaan ini ada dua, yaitu:

  • Game 8 bit berdasarkan style / art

    Jika berdasarkan gaya atau style berbasis pixel art. Maka game yang menggunakan grafik pixel ini dapat disebut sebagai game 8 bit. Tidak peduli game tersebut dirilis di platform dan konsol apa saja, baik itu game 2D maupun game 3D.

  • Game 8 bit berdasarkan prosesor

    Jika yang dimaksud sebagai game jadul yang dijalankan pada konsol berprosesor 8 bit. Seperti berbagai jenis game di konsol nes, famicom, dan sega sms. Maka game berbasis pixel art di era modern ini tidak bisa disebut sebagai game 8 bit.


Apakah Game 8 Bit Jadul Sama dengan Game 8 Bit Modern ?

Jika didasarkan pada cara pembuatan, penggunaan dan penempatan game. Maka game 8 bit jadul sama sekali tidak bisa disamakan dengan game yang memiliki art bergaya pixel di zaman modern. Hal ini dikarenakan adanya perbedaan didalam masalah mesin prosesor pada konsol game yang digunakan.

Sebut saja misalnya jika game super mario bros nes disebut sebagai game jadul 8 bit, maka game pixel dungeon, minecraft maupun game terraria di hp android tidak bisa disebut sebagai game 8 bit.


Basis Penggemar Game Berbasis 8 Bit

Apakah masih ada peminat dan penikmat game bergaya retro ala konsol jadul ?. Jika hal ini yang anda tanyakan, tentu sangatlah banyak. Karena orang tua kita, yang lahir pada era tahun 70-90 an masih sangat merindukan dan ingin bernostalgia dengan gaya game klasik jadul. Sebut saja misalnya seperti game sonic, tetris, super mario bros, bomberman, tank 1990, twin bee, ninja gaiden, excite bike, dan sejenisnya. Berbagai jenis retro game yang ada masih banyak memiliki peminat.

Dan faktanya peminat retro game ini bukan hanya didominasi oleh kalangan orang tua, bahkan anak muda pun banyak yang menyukai style dan art game berbasis pixel art ini. Apalagi dengan nuansa musik retro / chiptune yang melegenda. Pastinya masih sangat banyak peminatnya.


Apakah Game 8 Bit Masih Memiliki Masa Depan yang Cerah ?

Sudah terbukti adanya game berbasis pixel art ini. Sebut saja seperti game minecraft, super mario bros, tetris dan teman-temannya merajai angka penjualan game hingga detik ini. Dan game bergaya retro ini terbukti mengalahkan berbagai jenis pesaing yang mengusung tema game berbasis modern. Baik itu misalnya seperti game battle royale 3D, game fps, game open world, mmorpg dan sebagainya.

Jadi, bagi para pengembang game tidak perlu khawatir kehilangan pasar penjualan game berbasis pixel art. Karena game ini masih memiliki potensi penjualan yang sangat besar.
